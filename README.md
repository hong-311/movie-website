## 💡 주요 기능
⭐ 홈페이지
- 최신 영화 소식과 트레일러 표시
- 인기 있는 영화 목록 표시

⭐ 영화 상세 정보
- 영화 포스터, 제목, 개봉 날짜, 감독, 주요 배우, 장르, 줄거리 표시

⭐ 다양한 콘텐츠 제공
- 드라마, 영화, 예능 프로그램, 다큐멘터리, 웹드라마 등 다양한 콘텐츠를 시청할 수 있습니다.
